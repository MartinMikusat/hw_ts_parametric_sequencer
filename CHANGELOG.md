# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-11-18

### Added
- Initial release
- Core Sequencer class for animation playback
- Node system for declarative animation definitions
- Support for model animations (position, rotation, opacity)
- Camera animation support
- Hierarchical model positioning with markers
- Time reconciliation for relative timing dependencies
- Math utilities (Vector3, Quaternion, Euler)
- Smooth interpolation (linear for positions, SLERP for rotations)
